#### :zap: Startup

##### - Mockoon

- Install [Mockoon](https://mockoon.com/download/) API Mock and open **mockoonAPI.json**

- Start Mockoon Server

##### - Project

- Clone this project

- Install dependencies with `yarn` or `npm install`

- Run `yarn dev` or `npm run dev`

- Default Login

  * username: hello

  * password: 1234
